Function:

reverse every k nodes of a linked list.
If the list size is not a multiple of k, then leave the remainder nodes as is
 
Example:

Inputs:  1->2->3->4->5->6->7->8->NULL and k = 3

Output:  3->2->1->6->5->4->7->8->NULL
